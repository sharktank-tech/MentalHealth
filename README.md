# MentalHealth

O **MentalHealth** é uma plataforma de treino cognitivo para melhorar a performaçe mental.

## Demonstração
Acesse a aplicação: [mental-health-two-fawn.vercel.app]([(https://mental-health-two-fawn.vercel.app/))

---

## Funcionalidades

- **Dashboard:** Dashboard com metricas para avaliar o desempenho.
- **Rotina diária:** Rotina diária par aorganização do treino cognitivo.
- **Respiração:** Treino de respiração com técnica especializada.
- **Cálculo mental:** Treino de cálculo mental
- **Flashcards:** Treino de memória com flashcards.
- **Digitação:** Treino de digitação.
- **Sudoku:** Jogo de Sudoku.
- **Criatividade:** Treino de criatividade.

---

## Tecnologias Utilizadas

- **Backend:** Flask (Python)
- **Banco de Dados:** PostgreSQL
- **Frontend:** HTML, CSS e JavaScript
- **Hospedagem:** Vercel
- **Gerenciamento de Dependências:** Pip
- **Envio de Emails:** Flask-Mail

---

## Configuração do Projeto

### Pré-requisitos

- Python 3.9+
- PostgreSQL
- Gerenciador de pacotes `pip`

### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/sharktank-tech/MentalHealth
   cd MentalHealth
2. **Crie o ambiente virtual:**
   ```bash
    python3 -m venv .venv
3. **Atuve o ambiente:**
   ```bash
    source .venv/bin/activate
4. **Execute o arquivo principal para subir o servidor localmente:**
   ```bash
    python run.py
5. **Acesse sua aplicação:**
   http://127.0.0.0:5000
